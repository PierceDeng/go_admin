## 后端技术栈

* Go
* Gin
* Gorm
* Mysql
* Redis

启动前置：
1.  修改config包下的config.yaml文件
2. 导入config包下的sql文件
3. 运行即可

主要是帮助搭建后台管理系统，一个简易脚手架，很多东西可以自行实现


## 前端技术栈


这里是使用开源的RouYi框架，RuoYi v3.9.1版本，进行了细微的改版，删除了一些多余的功能。启动步骤请阅读ui包下的md文档

Github地址：[RuoYi-Vue](https://gitee.com/y_project/RuoYi-Vue)


## 内置功能

1.  用户管理：用户是系统操作者，该功能主要完成系统用户配置。
2.  菜单管理：配置系统菜单，操作权限，按钮权限标识等。
3. 后续再加！！！
